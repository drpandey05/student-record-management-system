# Student Record Management System

## About the Project
This project is a simple student record management system built using Python.  
It helps in storing, managing, and updating student records efficiently.

## Problem Statement
Managing student records manually is time-consuming and error-prone.  
This project provides an automated solution to handle student data easily.

## Features
- Add student records
- Display all records
- Search student by roll number
- Modify student details
- Delete student records
- Automatic percentage and grade calculation

## Technologies Used
- Python
- File Handling
